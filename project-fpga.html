Neural Networks on FPGA

My current deep dive into hardware acceleration.

Project Overview

In Progress

This project explores the intersection of hardware design and machine learning by implementing a feed-forward neural network for image classification directly onto an FPGA. The goal is to create a highly efficient, low-latency hardware accelerator for AI tasks.

By moving the computation from software to dedicated hardware, we can achieve significant performance gains, which is critical for real-time applications. The network architecture (784-30-30-10) is designed to classify handwritten digits from the MNIST dataset.

Current Focus & Learning Goals

  • Fixed-Point Arithmetic: I am currently implementing the neural network using fixed-point numbers to efficiently represent weights and biases in hardware, balancing precision with resource usage.
  • Hardware-Efficient Design: I am practicing RTL coding in Verilog with a strong focus on pipelining and creating a sequential datapath to maximize throughput and minimize latency.
  • Custom IP Core: The design plan includes a custom IP core with AXI-Stream and AXI-Lite interfaces, which will make the neural network a reusable component for future SoC designs.
  • Automation Framework: I am developing a Python framework to streamline the workflow, which will automate HDL generation, create testbenches, and run Vivado synthesis.
