5-stage Pipelined RISC Processor

Or, "How I learned to stop worrying and love the pipeline."

Project Overview

This project was a deep dive into computer architecture where I designed a 5-stage pipelined RISC processor from the ground up using Verilog. The core challenge was not just to build the stages—Instruction Fetch (IF), Decode (ID), Execute (EX), Memory (MEM), and Write Back (WB)—but to make them work together seamlessly.

A huge part of that was implementing crucial logic for hazard detection and data forwarding to prevent the pipeline from stalling, ensuring instructions flow through like a well-oiled machine.

Key Features

  • Full 5-stage RISC pipeline (IF, ID, EX, MEM, WB).
  • Hazard Detection Unit to identify data dependencies.
  • Data Forwarding logic to resolve hazards without stalling.
  • Modular Verilog design for components like the ALU, Control Unit, and pipeline registers.
  • Validated with a custom, cycle-accurate testbench to ensure correctness.

Verilog Modules

ALU

module ALU(
  input [31:0] A, B,
  input [1:0] ALUOp,
  output reg [31:0] Result
);
  always @(*) begin
    case (ALUOp)
      2'b00: Result = A + B;
      2'b01: Result = A | B;
      2'b10: Result = A - B;
      2'b11: Result = ~(A & B);
      default: Result = 32'b0;
    endcase
  end
endmodule

Control Unit

module ControlUnit(
  input [3:0] opcode,
  input ST,
  output reg ALUSrc, 
  output reg [1:0] ALUOp,
  output reg MR, MW, MReg, EnIM, EnRW
);
  always @(*) begin
    // Default values
    ALUSrc=0; ALUOp=2'b00; MR=0; MW=0;
    MReg=0; EnIM=1; EnRW=0;
    if (ST) begin
      // Stall state
    end else begin
      case (opcode)
        4'b0000: begin // SW
          ALUSrc = 1; MW = 1;
        end
        4'b0001: begin // ADD
          EnRW = 1;
        end
        4'b0011: begin // OR
          ALUOp = 2'b01; EnRW = 1;
        end
        4'b0111: begin // SUBI
          ALUSrc = 1; ALUOp = 2'b10; EnRW = 1;
        end
        4'b1111: begin // NAND
          ALUOp = 2'b11; EnRW = 1;
        end
      endcase
    end
  end
endmodule

Data Forwarding Unit

module ForwardUnit(
  input [3:0] EX_RN1, EX_RN2,
  input [3:0] MEM_WN, WB_WN,
  input MEM_RegWrite, WB_RegWrite,
  output reg [1:0] FA, FB
);
  always @(*) begin
    // Forward A
    if (MEM_RegWrite && (MEM_WN == EX_RN1))
      FA = 2'b10;
    else if (WB_RegWrite && (WB_WN == EX_RN1))
      FA = 2'b01;
    else
      FA = 2'b00;

    // Forward B
    if (MEM_RegWrite && (MEM_WN == EX_RN2))
      FB = 2'b10;
    else if (WB_RegWrite && (WB_WN == EX_RN2))
      FB = 2'b01;
    else
      FB = 2'b00;
  end
endmodule
