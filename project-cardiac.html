Cardiac Stress Detection

Using Multi-Modal Feature Fusion.

The Challenge

While traditional ECGs are powerful, their sensitivity can be limited when trying to detect the subtle, complex changes caused by psychological stress. To overcome this, this project explored a multi-modal fusion approach, integrating data from ECG, Heart Rate Variability (HRV), and Photoplethysmography (PPG) to create a more holistic and accurate assessment of cardiac stress.

Methodology

  • Developed a complete signal processing pipeline in MATLAB using both real and synthetic ECG data.
  • Applied a 4th-order Butterworth bandpass filter to remove noise and isolate relevant frequency components.
  • Implemented R-Peak detection using a `findpeaks()` algorithm to identify individual heartbeats.
  • Calculated RR intervals and extracted key HRV metrics like SDNN, RMSSD, and the LF/HF ratio for stress classification.
  • Utilized multi-modal fusion to combine the physiological markers from each signal, significantly enhancing sensitivity and reliability.

Key Outcome

The multi-modal approach was a success, improving stress classification accuracy from:

~70%

(ECG-only)

↓

Over 90%

(Multi-Modal Fusion)

MATLAB Analysis Code

Signal Processing Snippet

% Generate or load ECG signal
fs = 360; % Sampling rate
[b, a] = butter(4, [0.5 40] / (fs/2), 'bandpass');
filt_ecg = filtfilt(b, a, ecg_signal);

% R-Peak Detection
[pks, locs] = findpeaks(filt_ecg, 'MinPeakHeight', 0.8);

% HRV Analysis
rr_intervals = diff(locs) / fs;
sdnn = std(rr_intervals) * 1000;
rmssd = sqrt(mean(diff(rr_intervals).^2)) * 1000;

% Frequency Domain
f_interp = 250;
t_interp = 0:1/f_interp:rr_time(end);
rr_interp = interp1(rr_time, rr_intervals, t_interp, 'pchip');
lf_power = bandpower(rr_interp, f_interp, [0.04 0.15]);
hf_power = bandpower(rr_interp, f_interp, [0.15 0.4]);
lf_hf_ratio = lf_power / hf_power;

if lf_hf_ratio > 2.0
    disp('High Stress');
else
    disp('Normal Stress');
end
